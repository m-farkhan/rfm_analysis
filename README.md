# rfm_analysis
End-to-end customer segmentation project using Python, SQL, Excel, and Tableau. Analyzed UK Online Retail data to identify high-value customer segments and optimize retention strategies. Includes data cleaning, EDA, RFM segmentation analysis, and interactive dashboard visualization.
